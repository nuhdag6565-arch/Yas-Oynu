# 🎯 Şanslı Yaş Çekilişi (Lucky Age Lottery)

Bu proje, Java ile yazılmış eğlenceli ve basit bir konsol tabanlı şans oyunudur. Sistem, belirlenen yaş aralığında (15-60) rastgele bir "şanslı yaş" belirler ve kullanıcıdan yaşını girerek bu şanslı yaşı bulmasını bekler.

## ✨ Özellikler

- **Rastgele Sayı Üretimi:** Sistem her çalıştığında `Math.random()` kullanılarak 15 ile 60 arasında rastgele bir yaş belirlenir.
- **Kullanıcı Girdisi Doğrulama:** Kullanıcının harf veya geçersiz bir karakter girmesi durumunda programın çökmesini engellemek için `Scanner.hasNextInt()` ile güvenlik kontrolü yapılır.
- **Yaş Aralığı Kontrolü:** Sadece 15-60 yaş aralığındaki kullanıcıların katılımına izin verilir. Kurallara uymayan girişlerde uyarı mesajı gösterilir.

## 🛠️ Kullanılan Teknolojiler

- Java (JDK)
- Temel Java Kütüphaneleri (`java.util.Scanner`, `java.lang.Math`)
